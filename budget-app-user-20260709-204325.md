# Budget App — User Summary

**Project:** Personal Budget Tracking App (记账软件)
**File:** `/home/xbl2602/budget-app.html`
**Language:** Chinese (UI), Currency: MYR (RM)

---

## What Was Built

A complete single-page budget tracking application in one HTML file. Open it in any modern browser — no installation or internet required.

## Features

### 📊 Dashboard (总览)
- Monthly spending overview with 4 summary cards
- Budget and savings progress rings (canvas)
- 7-day spending trend sparkline
- Top 5 spending categories this month
- Overspend warnings (amber at 80%, red at 100%)
- Savings prediction with daily remaining limit

### ➕ Add Record (记账)
- Amount input with RM prefix
- Category selector with tree picker (emoji + color)
- Date/time picker (defaults to now)
- Notes field (up to 200 characters)
- Validation and success toast

### 📋 Records (流水)
- Card list sorted newest-first
- Tap card to edit, swipe to delete
- Advanced search: keyword, date range, category filter, amount range
- Results count and infinite scroll

### 🏷️ Categories (分类)
- 8 root categories with 21 sub-categories pre-loaded
- Recursive tree with expand/collapse
- Add root/sub categories
- Rename, change emoji icon (grid picker), change color (palette + custom HEX)
- Move category with cycle prevention
- Delete with cascade options (delete children or move to parent)

### 📈 Statistics (统计)
- Month selector or custom date range
- Summary cards for total, daily average, prediction, savings prediction
- Interactive pie chart (click to drill down into sub-categories)
- Smooth-curve line chart with gradient fill (daily trend)
- Color-coded bar chart (category comparison, click to drill down)
- All charts have "Download PNG" buttons

### ⚙️ Settings (设置)
- Monthly budget setting
- Savings target (fixed amount, percentage of budget, or both)
- Export/Import JSON (replace or merge modes)
- Export CSV
- Clear all data (with confirmation)

## Technical Details

- **File:** Single `.html` file (104 KB, 2895 lines)
- **Dependencies:** Zero — no CDN, no libraries, no frameworks
- **Storage:** All data saved in your browser's localStorage
- **Responsive:** Mobile-first design with bottom tab nav (mobile) and sidebar (desktop)
- **Breakpoints:** 480px (tablet), 1024px (desktop)
- **Routing:** Hash-based (`#/overview`, `#/add`, etc.)
- **Data Models:** Records (with category linking), Categories (infinite nesting), Budgets (per month), Savings Targets

## Known Notes

- The app requires a modern browser (ES2020+). Internet Explorer is not supported.
- Data is stored in your browser's localStorage — clearing your browser data will lose your records. Use the Export JSON feature to backup your data.
- Charts (pie, line, bar) use HTML5 Canvas and support "Download PNG" for saving.
