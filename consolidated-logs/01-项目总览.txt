## 六个核心页面

### 1. 总览

提供月度财务概览的首页。

**摘要卡片（4列网格）：**
- 本月总支出 — 当月总支出金额
- 预算 — 月度预算金额（未设置时显示"未设置"）
- 日均支出 — 日均支出金额
- 预测月总支出 — 基于日均支出预测的月末总金额

**预算进度环：**
- Canvas 环形图，显示可支配预算已使用的百分比
- 可支配预算 = 月度预算 - 储蓄目标
- 超支时变红，并显示超支金额
- 下方显示等式："零花钱 RM1500 - 储蓄 RM500 = 可支配 RM1000"

**储蓄目标环：**
- Canvas 环形图，跟踪实际储蓄与目标储蓄的进度
- 达成时显示绿色（显示"已达成"），进行中显示琥珀色，未设置时显示灰色

**储蓄预测卡片：**
- 四列明细：当前已存金额、储蓄目标、预测月末储蓄、目标达成率
- 双每日剩余限额：总额（扣除储蓄前）和净额（扣除储蓄后）
- 智能提示消息：正面预测（"预计月末可存"）或警告（"预计超支"）

**其他模块：**
- 最近7天趋势图：迷你 Canvas 折线图，显示每日支出波动
- 相比上月：环比变化百分比
- 支出 Top 5：本月支出排名前列的分类
- 预算进度卡片：可复用组件，支持整体/分段切换、排序、监控选择
- 超支警告：红色边框卡片，列出超过预算 80% 的分类
- 快捷操作："记一笔"和"查看统计"按钮

**空状态：** 无记录时显示带虚线边框和渐变背景的欢迎卡片。

### 2. 记账

**表单字段：**
- 金额：文本输入框，带 RM 前缀，自动过滤非数字字符，最多2位小数，支持手机数字键盘
- 分类：按钮，打开树形分类选择器弹窗，选中后显示颜色圆点 + 图标 + 名称
- 日期/时间：datetime-local 输入框，默认为当前本地时间并带时区偏移
- 备注：文本输入框，最多200字符，占位符"例：椰浆饭"
- 提交：验证金额 > 0 且已选择分类；验证错误时抖动动画；成功后重置

**分类选择器：** 可复用的树形弹窗，用于7种以上场景（添加、编辑、筛选、批量、合并、移动）。

---

### 3. 流水

**高级筛选（多条件）：**
- 关键词搜索（搜索备注，不区分大小写）
- 分类筛选（树形选择器，含"全部分类"选项，按后代 ID 过滤）
- 日期范围（开始 + 结束日期输入）
- 金额范围（最小 + 最大数值输入）
- 仅显示超支项开关
- 一键清除所有筛选

**两种视图模式：**

| 卡片视图（默认） | 紧凑视图 |
|---|---|
| 大圆形图标 + 分类名称垂直布局 | 日期、分类、备注、金额在同一水平行 |
| 日期、备注分多行显示 | 长文本备注自动截断 |
| 触摸滑动删除 | 删除按钮始终可见 |
| 适合浏览详情 | 适合快速浏览多条记录 |

偏好设置自动保存至 localStorage。

**记录操作：**
- 点击编辑：打开弹窗，预填金额、分类、日期、备注
- 删除：桌面端悬停显示删除按钮；移动端滑动显示删除；弹窗中也有删除选项
- 无限滚动：自动加载更多记录
- 记录计数：实时显示筛选后总数

**数据导出：**
- Excel 导出（XML Spreadsheet 2003，含活动公式，5个工作表）
- CSV 导出（UTF-8 BOM）
- JSON 导出/导入（备份和迁移）

**批量操作：**
- 批量删除（每条记录支持软删除和撤销）
- 批量更改分类
- 多选模式，每条记录带复选框

### 4. 分类

**无限嵌套：** 支持任意深度的分类层级。

**预设分类：** 8个根分类，22个子分类：

| 根分类 | 子分类 |
|---|---|
| 餐饮 | 早餐, 午餐, 晚餐, 饮料/咖啡 |
| 交通 | 油费, 停车, 公交/地铁, 打车 |
| 购物 | 日用品, 服饰, 电子产品 |
| 娱乐 | 电影, 游戏, 运动 |
| 居住 | 房租, 水电, 网络 |
| 医疗 | 看病, 药品 |
| 教育 | 书籍, 课程 |
| 其他 | （无） |

**每个分类的操作：**
- 添加子分类：提示输入名称
- 重命名：弹窗文本输入
- 更改图标：表情符号图标网格（可滚动）
- 更改颜色：14个预设色块 + 自定义 HEX 输入
- 移动：树形选择器选择新父分类，含循环检测
- 合并：合并到另一分类（记录重新分配，子分类处理选项）
- 删除：级联选项（将子分类移至父分类、删除子分类或取消）
- 展开/折叠：箭头切换，带 CSS 动画

**每个分类的预算：**
- 树形标题行中的内联输入框（始终可见）
- RM / % 切换，在固定金额和百分比之间切换
- 百分比基于月度总预算计算
- 父子分类预算校验：子分类总和不能超过父分类

**手风琴设计：**
- 简洁手风琴布局，包含标题行 + 可折叠内容区
- 所有分类默认折叠（不自动展开）
- 每个标题显示：箭头、颜色圆点、表情符号图标、名称、操作图标（+ 和设置）、预算输入
- 深度 >= 4 时显示层级路径
- "全部展开"/"全部折叠"链接

### 5. 统计

**日期选择：**
- 月份选择器（`<input type="month">`）
- 自定义日期范围（两个日期输入框，开始日期超过结束日期时自动互换）
- 两种模式互斥

**摘要卡片（第一行）：**
- 总支出、日均支出（自定义范围为记录数）、预测总支出、储蓄预测（绿色/红色标记）

**交易分析卡片（第二行）：**
- 交易笔数 + 有交易的天数
- 每笔交易平均值
- 单笔最大金额 + 分类
- 单日最高支出 + 日期

**储蓄估算卡片（仅月度模式）：**
- 预算、当前已存、预测储蓄、盈余
- 双每日限额（总额/净额）

**图表（均基于 Canvas，均可下载为 PNG）：**

| 图表 | 类型 | 交互 |
|---|---|---|
| 日历热力图 | HTML 网格（6色渐变） | 点击日期查看记录，可展开 |
| 饼图 | Canvas（弹性动画） | 悬停突出显示，点击下钻 |
| 折线图 | Canvas（800ms 缓出） | 贝塞尔曲线动画，渐变填充 |
| 预算进度卡片 | HTML 组件 | 整体/分段切换，排序，监控 |
| 月度环比对比 | Canvas 分组柱状图 | 可切换 |
| 月度支出趋势 | Canvas 折线图（6个月） | 渐变填充，数据点，标签 |
| 月度储蓄趋势 | Canvas 柱状图（6个月） | 绿色/红色柱子，零线 |

**交互功能：**
- 饼图下钻：点击切片 → 子分类；面包屑导航；返回按钮
- 柱状图下钻：点击柱子 → 子分类
- 图表展开：按钮打开放大视图的弹窗
- 图表 PNG 下载：每个图表都有下载按钮
- 热力图日期点击：显示该日期的记录

## 技术架构

```
budget-app.html
├── CSS（约1,050行）
│   ├── 设计系统（CSS 变量）
│   ├── 排版与网格
│   ├── 组件样式（按钮、卡片、表单、导航）
│   ├── 记录卡片与紧凑视图
│   ├── 分类手风琴
│   ├── 图表展开弹窗
│   ├── 动画与过渡
│   ├── 响应式适配（480px/1024px 断点）
│   └── 打印样式（用于报告页面）
├── HTML（约60行）
│   ├── Toast 通知容器
│   ├── 弹窗对话框
│   ├── 侧边栏导航（桌面端）
│   ├── 底部标签导航（移动端）
│   └── 7个页面容器
└── JavaScript（约5,200行）
    ├── 工具函数（uuid、formatMoney、escHtml）
    ├── DataStore（CRUD + localStorage 持久化，约300行）
    ├── StatsEngine（数据分析引擎，约160行）
    ├── Excel 导出（XML Spreadsheet 生成器，约460行）
    ├── 7个页面渲染函数
    ├── Canvas 图表绘制（饼图/折线图/柱状图/环形图/趋势图，约900行）
    ├── 分类树管理
    ├── 记录筛选与分页
    ├── 弹窗与 Toast
    └── 事件绑定与路由
```

### 关键技术决策
- **零外部依赖：** 无第三方库，降低安全风险和加载时间
- **Canvas 图表：** 所有图表使用原生 Canvas API，避免 SVG/DOM 性能开销
- **XML Spreadsheet 2003：** Excel 导出使用原始 XML 格式，无需 ZIP 库
- **使用 setTransform 代替 scale：** 防止多次重绘时变换矩阵累积
- **CSS 控制 Canvas 尺寸：** 移除 JS 对 `canvas.style.width` 的覆盖，避免布局反馈循环
- **哈希路由：** `location.hash` 驱动标签切换，支持浏览器前进/后退
- **requestAnimationFrame 动画：** 所有图表动画使用 rAF 实现平滑逐帧渲染
- **DPR 感知尺寸：** Canvas 尺寸考虑 `window.devicePixelRatio`，实现高 DPI 清晰度

### 导航（7个标签）

| 标签 | 名称 | 函数 |
|---|---|---|
| `overview` | 总览 | `renderOverview()` |

## 快速入门指南

1. 在现代浏览器中打开 `budget-app.html`

## 使用建议

- **先设置预算：** 配置月度零花钱，以启用预算环和储蓄预测功能
- **建立分类体系：** 创建符合个人消费习惯的分类树（例如"餐饮 -> 午饭/晚餐/零食"）
- **即时记账：** 每次消费后立即记录，避免遗忘
- **每周复盘：** 每周查看统计页面，分析消费模式
- **定期备份：** 每月导出 JSON 或 Excel，确保数据安全

## Excel 导出详情

生成的 Excel 文件包含 5 个工作表，均带活动公式：

| 工作表 | 内容 | 公式示例 |
|---|---|---|
| 消费记录 | 所有交易详情及汇总 | `=SUM()`、`=AVERAGE()`、`=MAX()`、`=MIN()` |
| 分类统计 | 根分类及子分类汇总与占比 | `=SUMIF()`、`=IF(..., /, 0)` |
| 月度统计 | 月度汇总：支出、均值、预算、可支配、储蓄、使用率 | `=AVERAGE()`、`=IF(RC[-1]>0, RC[-1]-RC[-5], ...)` |
| 预算跟踪 | 预算 vs 实际：剩余、使用率、自动状态 | `=IF(RC[-1]>=0, "正常", "超支")` |
| 储蓄统计 | 月度储蓄：实际、目标、达成率 | `=MIN(1, MAX(0, RC[-1]/RC[-4]))` |

功能：冻结表头、自动筛选启用、货币/百分比格式设置、兼容 Excel/WPS/LibreOffice/Google Sheets。

## 交互详情

### 饼图悬停高亮
悬停在饼图切片上时：
1. 切片沿中心轴弹出 10px
2. 颜色加深 18%
3. 添加投影（浮动效果）
4. 添加白色边框描边
5. 光标变为手型指针

### 动画效果
- 饼图：600ms 弹性弹跳动画（1 - pow(2, -7x) * cos(x * PI * 2 * 1.2)）
- 折线图：800ms 缓出 cubic-bezier 绘制
- 记录卡片：悬停微交互（上移 1px + 阴影加深）
- 页面切换：350ms 滑动 + 缩放
- Toast：类似弹性的 cubic-bezier 弹跳
- 弹窗：过冲弹簧曲线 + 背景模糊
- 图表展开：250ms 淡入 + 滑动

### 日期范围自动修正
当自定义结束日期早于开始日期时，系统自动互换两者并显示"日期范围已自动修正"。

### 金额输入智能过滤
- 仅允许输入数字和小数点
- 最多2位小数
- 使用 `oninput`（而非 `onkeypress`）避免阻塞键盘输入
