## Six Core Pages

### 1. Overview (Zong Lan / 总览)

The landing page providing a monthly financial snapshot.

**Summary Cards (Grid-4):**
- Ben Yue Zong Zhi Chu (本月总支出) — total spending for the current month
- Yu Suan (预算) — monthly budget amount (shows "未设置" if unset)
- Ri Jun Zhi Chu (日均支出) — daily average spending
- Yu Ce Yue Zong (预测月总支出) — predicted end-of-month total based on daily average

**Budget Progress Ring:**
- Canvas donut chart showing percentage of spendable budget consumed
- Spendable budget = Monthly budget - Savings target
- Turns red when overspent, showing overspend amount
- Equation displayed below: "零花钱 RM1500 - 储蓄 RM500 = 可支配 RM1000"

**Savings Target Ring:**
- Canvas donut tracking actual savings vs target savings progress
- Green when achieved (显示 "已达成"), amber when in progress, gray when unset

**Savings Prediction Card:**
- Four-column breakdown: current saved, savings target, predicted end-of-month savings, goal achievement rate
- Dual daily remaining limits: gross (before savings deduction) and net (after savings deduction)
- Smart narrative message: positive prediction ("预计月末可存") or warning ("预计超支")

**Other Modules:**
- Last 7 Days Sparkline: mini canvas line chart showing daily spending fluctuation
- Mo Bi (相比上月): month-over-month change percentage
- Spending Top 5: ranked categories by spending this month
- Budget Progress Card: reusable component with solid/segmented toggle, sort, monitor selection
- Overspend Warning: red-bordered card listing categories exceeding 80% of budget
- Quick Actions: "记一笔" (add record) and "查看统计" (view stats) buttons

**Empty State:** Welcome card with dashed border and gradient background when no records exist.

### 2. Add Record (Ji Yi Bi / 记一笔)

**Form Fields:**
- Amount: text input with RM prefix, auto-filters non-numeric characters, max 2 decimal places, mobile numeric keyboard
- Category: button opening tree category picker modal, shows color dot + icon + name when selected
- Date/Time: datetime-local input, defaults to current local time with timezone offset
- Notes: text input, max 200 characters, placeholder "例：椰浆饭"
- Submit: validates amount > 0 and category selected; shake animation on validation error; resets on success

**Category Picker:** Reusable tree modal used across 7+ contexts (add, edit, filter, batch, merge, move).

### 3. Records (Liu Shui / 流水)

**Advanced Filters (Multi-condition):**
- Keyword search (searches notes, case-insensitive)
- Category filter (tree picker with "All categories" option, filters by descendant IDs)
- Date range (start + end date inputs)
- Amount range (min + max number inputs)
- Overspent-only toggle
- One-click clear all filters

**Two View Modes:**
| Card View (default) | Compact View |
|---|---|
| Large circular icon + category name vertical layout | Date, category, note, amount in single horizontal row |
| Date, notes on separate lines | Note auto-truncated for long text |
| Touch swipe to delete | Delete button always visible |
| Best for browsing details | Best for scanning many records |

Preference auto-saved to localStorage.

**Record Operations:**
- Click to edit: opens modal with amount, category, date, note pre-filled
- Delete: desktop hover shows trash button; mobile swipe reveals delete; modal also has delete
- Infinite scroll: auto-loads more records
- Record count: real-time filtered total

**Data Export:**
- Excel export (XML Spreadsheet 2003 with live formulas, 5 sheets)
- CSV export (UTF-8 BOM)
- JSON export/import (backup and migration)

**Batch Operations:**
- Batch delete (soft delete with undo for each)
- Batch change category
- Multi-select mode with checkbox per record

| 其他 | (none) |

**Per-Category Actions:**
- Add child: prompts for name
- Rename: modal text input
- Change icon: grid of emoji icons (scrollable)
- Change color: 14 preset color swatches + custom HEX input
- Move: tree picker for new parent with cycle detection
- Merge: merge into another category (records reassigned, child handling options)
- Delete: cascade options (move children to parent, delete children, or cancel)
- Expand/Collapse: arrow toggle with CSS animation

**Per-Category Budget:**
- Inline input in tree header row (always visible)
- RM / % toggle to switch between fixed amount and percentage
- Percentage computed as % of total monthly budget
- Parent-child budget validation: child sum cannot exceed parent

**Accordion Design:**
- Clean accordion (手风琴) layout with header row + collapsible body
- All categories start collapsed (no auto-expand)
- Each header shows: arrow, color dot, emoji icon, name, action icons (+ and settings), budget input
- Deep nesting breadcrumb at depth >= 4
- "Expand All" / "Collapse All" links

### Key Technology Decisions
- **Zero External Dependencies:** No third-party libraries, reducing security risks and load time
- **Canvas Charts:** Native Canvas API for all charts, avoiding SVG/DOM performance overhead
- **XML Spreadsheet 2003:** Excel export uses raw XML format, requiring no ZIP library
- **setTransform instead of scale:** Prevents transform matrix accumulation on multiple redraws
- **CSS-controlled canvas sizing:** Removes JS override of `canvas.style.width` to avoid layout feedback loop
- **Hash routing:** `location.hash` drives tab switching, supports browser back/forward
- **requestAnimationFrame animation:** All chart animations use rAF for smooth frame-by-frame rendering
- **DPR-aware sizing:** Canvas dimensions account for `window.devicePixelRatio` for high-DPI clarity

## Quick Start Guide

1. Open `budget-app.html` in any modern browser
2. Go to Settings (设置) page to set monthly allowance and savings target
3. Go to Add Record (记一笔) to start logging expenses
4. Check Overview (总览) and Statistics (统计) regularly to monitor finances
5. Export Excel or JSON for data archiving

---

## Usage Suggestions

- **Set a budget first:** Configure monthly allowance to enable budget ring and savings predictions
- **Build a category system:** Create category tree matching personal spending habits (e.g., "餐饮 -> 午饭/晚餐/零食")
- **Log immediately:** Record expenses right after each purchase to avoid forgetting
- **Review weekly:** Check the Statistics page weekly to analyze spending patterns
- **Backup regularly:** Export JSON or Excel monthly for data safekeeping

## Excel Export Details

The generated Excel file contains 5 worksheets with live formulas:

| Sheet | Content | Formula Examples |
|---|---|---|
| 消费记录 (Records) | All transaction details with totals | `=SUM()`, `=AVERAGE()`, `=MAX()`, `=MIN()` |
| 分类统计 (Category Stats) | Root + sub-category totals and ratios | `=SUMIF()`, `=IF(..., /, 0)` |
| 月度统计 (Monthly Stats) | Monthly summaries: spending, avg, budget, disposable, savings, usage rate | `=AVERAGE()`, `=IF(RC[-1]>0, RC[-1]-RC[-5], ...)` |
| 预算跟踪 (Budget Tracking) | Budget vs actual: remaining, usage rate, auto status | `=IF(RC[-1]>=0, "正常", "超支")` |
| 储蓄统计 (Savings Stats) | Monthly savings: actual, target, achievement rate | `=MIN(1, MAX(0, RC[-1]/RC[-4]))` |

Features: Frozen headers, auto-filter enabled, currency/percentage formatting, compatible with Excel/WPS/LibreOffice/Google Sheets.

## Interaction Details

### Pie Chart Hover Highlight
On hover over a pie slice:
1. Slice pops out 10px along center axis
2. Color deepens by 18%
3. Drop shadow added (float effect)
4. White border stroke added
5. Cursor changes to pointer

### Animations
- Pie chart: 600ms elastic spring animation (1 - pow(2, -7x) * cos(x * PI * 2 * 1.2))
- Line chart: 800ms ease-out cubic bezier draw
- Record cards: hover micro-interaction (1px lift + shadow deepen)
- Page transitions: 350ms slide + scale
- Toast: spring-like cubic-bezier with bounce
- Modal: overshoot spring curve + backdrop blur
- Chart expand: 250ms fade + slide

### Date Range Auto-Correction
When custom end date is earlier than start date, the system auto-swaps them and displays "日期范围已自动修正".

### Amount Input Smart Filtering
- Only digits and decimal point allowed
- Max 2 decimal places
- Uses `oninput` (not `onkeypress`) to avoid blocking keyboard input
