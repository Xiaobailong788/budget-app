# 统计页新增 Waffle Chart 可视化卡片

## 变更摘要

### 已移除
- 旧的"标签筛选"卡片（文本标签选择器）
- 相关的 `statsTagFilter` 变量和过滤逻辑
- `openTagPickerForStats()` 和 `removeStatsTagFilter()` 函数

### 新增功能

**🏷️ 标签分布 (Waffle Chart) 卡片** — 位于统计页最底部（"月度储蓄趋势"之后）

1. **Waffle Chart 可视化**：每个小方块代表一定金额，按标签着色，直观展示各标签的支出占比
2. **5 档密度选择器**：点击 █ 数量可切换网格密度（50~500 个单元格），精度越高细节越丰富
3. **"含无标签"开关**：点击切换是否包含未打标签的记录（灰色显示）
4. **Scale Pop 动画**：页面加载时单元格会依次弹性弹出，视觉效果流畅
5. **悬停交互**：鼠标移到单元格上会显示标签名称、金额和百分比，同时高亮所有同色单元格

### 操作说明
- **切换密度**：点击卡片右上角的 ██ 图标（1-5 格），密度越高单元格越多
- **包含无标签**：点击"含无标签"开关切换
- **查看详情**：鼠标悬停在任意单元格上查看标签详情
- 所有设置自动保存，下次打开页面时恢复
